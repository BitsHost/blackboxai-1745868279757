Introduction to upMVC

upMVC is a powerful, modular MVC system that empowers your PHP development without the complexity of traditional frameworks. It provides a structured, straightforward approach while maintaining the flexibility to integrate modern technologies.

"Many frameworks may look very appealing at first glance because they seem to reduce web application development to a couple of trivial steps, but these same shortcuts are likely to be your bottlenecks since they achieve this simplicity by sacrificing flexibility and performance."

- Rasmus Lerdorf

Getting Started

Installation

# Install as a library
composer require bitshost/upmvc:dev-main

# Or install as a project
composer create-project bitshost/upmvc:dev-main yourFolderNameHere

Basic Configuration

After installation, you'll need to configure these key files:

  • /etc/Config.php - General configuration
  • /etc/ConfigDatabase.php - Database settings
  • /modules/mail/MailController.php - Mail configuration

Core Concepts

MMVC Architecture

MMVC (Modular Model View Controller) is built on these key principles:

Modularity

Work on modules independently without affecting the rest of your project

Language Freedom

Write modules in your preferred language - PHP, JS, Python, or modern frameworks

Working with Modules

Modules are the building blocks of your application. Each module follows the MVC pattern and can be developed independently.

Pro Tip: Use the BaseModel for quick CRUD operations - simply extend it in your module model.

Routing System

upMVC provides a flexible routing system with support for:

  • General Routes (/etc/Routes.php)
  • Module-specific Routes (/modules/yourmodule/routes/Routes.php)
  • Friendly URLs and aliases
  • Parameter handling

Modern Framework Integration

upMVC supports the "Islands Architecture" - allowing you to integrate modern frameworks like React, Vue, Preact or Svelte within your PHP application.

Islands of Interactivity

Strategically place interactive components built with modern frameworks within your PHP-generated HTML.
